# 자바 스프링 프레임워크(renew ver.)

- 주요기능
  - DI(Dependenct Injection)
  - AOP
  - MVC
  - JDBC

- 제공하는 모듈
  - `spring-core`: DI와 IoC(Inversion of Control) 제공
  - `spring-aop`: AOP구현 기능
  - `spring-jdbc`: 데이터베이스 조작 기능
  - `spring-tx`: 트랜잭션 관련 기능
  - `spring-webmvc`: 스프링 MVC 구현 기능

- 컨테이너: Spring에서 객체를 생성하고 조립
  - 빈: 컨테이너를 통해 생성된 객체
